# Usage
<!-- usage -->
```sh-session
$ npm install -g content-cli
$ content-cli COMMAND
running command...
$ content-cli (-v|--version|version)
content-cli/0.0.0 darwin-x64 node-v14.15.0
$ content-cli --help [COMMAND]
USAGE
  $ content-cli COMMAND
...
```
